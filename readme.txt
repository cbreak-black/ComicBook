ComicBook is a comic book reader program written for Mac OS X 10.8. It supports
individual files as well as archives containing images.

The GUI is implemented with CoreAnimation, image loading is multi-threaded with
GCD, archive support uses XADMaster from The Unarchiver. Application Sandboxing
is supported. The current position within a comic is saved persistently.
